# ⚙️ Execution & Progress System

---

# 1. Daily Loop (Progress-Based)

Repeat until section is complete:

### Step 1 — Learn

- Watch 1–2 videos
- Focus:
  - When to use the pattern
  - Why it exists
  - Recognition signals

---

### Step 2 — Solve Problems

- Solve 2–4 problems
- Stop when mentally fatigued (not time-based)

---

### Step 3 — Deep Understanding

For each problem:

- What pattern is this?
- What triggered it?
- What mistake did I make?

---

### Step 4 — Rebuild From Scratch

- No copying
- Must reconstruct solution independently

---

# 2. Retention System

Each problem has a level:

### 🟥 Level 1 — Just Learned

- Needed help

### 🟨 Level 2 — Semi Comfortable

- Can solve with effort

### 🟩 Level 3 — Mastered

- Can solve quickly (<10 min)

---

# 3. Daily Review Rule

At the start of each session:

- Re-solve:
  - 1 Level 1 problem
  - 1 Level 2 problem

---

# 4. Promotion Logic

Move problems:

- Level 1 → Level 2 → Level 3

If you fail during review:

- Drop one level

---

# 5. Section Completion Gate

You can move forward ONLY if:

### ✔ Recognition

- Identify pattern quickly

### ✔ Execution

- Solve medium problems

### ✔ Explanation

- Can teach the pattern clearly

---

# 6. Key Principle

This system is:

- Not time-based
- Not problem-count-based

It is:

> Mastery-based progression
